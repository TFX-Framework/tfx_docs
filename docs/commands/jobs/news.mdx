Set your position in the news job and assign props.

Give cash to the provided player.

###### Parameters
Argument | Type | Description
--- | --- | ---
ID | Number | The users Player ID.
AMOUNT | Number | Amount of money to give. 

## Usage
```
/news setrole {camera/mic/boom}
```

## Toggle Props
```
/news toggle
```
