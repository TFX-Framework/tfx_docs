# About

Not only are we looking to provide a one of a kind Roleplay Framework we are also actively working on other projects such as a Community CAD/MDT, Loggin Scripts, Bots and much more.

Support here is offered for FiveM as a Whole, if we know our way around the script you need help with we will be happy to help.

# ToxicFX Framework
**TFX** is a roleplay aimed framework for **[FiveM](https://fivem.net/)**. It features many resources to help people build the best roleplay servers.

---

## Features
- Notification system
- Economy and Bank system
- Identity system
- Garage and car seller system

> *This list is non-exhaustive and could change/be improved at any moment.*
